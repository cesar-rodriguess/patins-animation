# Snitap Patins

Uma landing page estática para a marca `Snitap`, desenvolvida com HTML e CSS modernos. O projeto apresenta um layout visual atraente com animações suaves, galeria e seções de promoção para demonstrar um conceito de e-commerce de patins.

## O que o projeto faz

`patins-animation` é uma página de apresentação estática que demonstra um design de interface para vendas de patins. Ele inclui:

- uma seção hero com chamadas para ação e animações de texto
- um carrossel de banner animado
- uma galeria de fotos com transições e efeitos hover
- navegação simples e rodapé com links de redes sociais

## Por que o projeto é útil

Este projeto é útil para desenvolvedores e designers que desejam:

- ver um exemplo de landing page estática com HTML e CSS puro
- aprender com animações CSS modernas e componentes visuais responsivos
- reutilizar estrutura e estilos para landing pages de produtos
- prototipar rapidamente um site promocional sem dependências de JavaScript

## Como começar

### Pré-requisitos

Nenhuma dependência de build é necessária. O projeto roda diretamente em qualquer navegador moderno.

### Executando localmente

1. Clone o repositório:

```bash
git clone <url-do-repositorio>
cd patins-animation
```

2. Abra `index.html` no navegador.

### Alternativa: servidor local

Se preferir usar um servidor local para desenvolvimento:

```bash
cd patins-animation
python -m http.server 8000
```

Abra `http://localhost:8000` no navegador.

### Estrutura de arquivos

- `index.html` — página principal
- `styles/` — estilos CSS divididos por seção
- `assets/` — imagens, ícones e elementos gráficos
